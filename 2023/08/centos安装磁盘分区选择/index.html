CentOS安装磁盘分区介绍

Device Type(设备类型)

设备类型说明
标准分区标准 分区可包含文件系统或swap空间，同时也可以为软件RAID或者LVM物理卷提供容器。
逻辑卷 LVM创建LVM分区可自动生成LVM逻辑卷。便于扩容。
LVM 精简配置使用精简配置，用户可以在程序需要的时候，动态扩展分配有效存储空间。
BTRFSBTRFS是多个类设备功能的文件系统，相比ext2、ext3和ext4，它可以处理和管理更多文件、更大型的文件以及更大的卷。

分区配置参考

分区所需容量说明（单位默认为MB）
/boot>=200MB用于存放引导器（bootloader）和内核映像（kernel image），必选
swap4096MB交换分区，可选
/剩余全部空间大小为空，默认会分配剩余空间

File Systom(文件系统)

需要勾选旁边的重新格式化。

类型最大支持文件系统最大文件最大子目录
ext2

ext41EB（1048576TB）16TB无限制
xfs16EB（约1600百万TB）千百万条目
